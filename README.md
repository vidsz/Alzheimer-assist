# Alzheimer's Assist

## Description
An android application to solve the issue of inconvenience faced by Alzheimer's patients by developing the app from a mental health perspective.

## Key Features
- Providing information and photos of family members
- Task list
- Maintaining a diary to help store their thoughts and memories
- Emergency contacts
- Booking appointments
- Daily Reminders

## Tech Stack
- Dart
- Flutter
- Firebase
- Coingecko API
